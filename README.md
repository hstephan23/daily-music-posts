# daily-music-post


## Description
The Daily Music Post application allows users to input their song of the day that other users can submit a comment and react to.

Built on:
* bcrypt
* connect-session-sequelize
* dotenv
* express
* express-handlebars
* express-session
* mysql2
* sequelize
